Web development is a course that gives students experience with frontend and backend technologies, as well as provides knowledge and insight into, among other things, Universal Design (UU) and usability. In the frontend part, students will learn both the basic techniques in web development with regard to HTML, CSS and JavaScript, as well as be able to use libraries and frameworks. In the backend part, the focus is on creating an API with a database that can be consumed from the frontend. One should create responsive web applications that ensures the interests of a user (ref. UU and usability).
